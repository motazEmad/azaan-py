# Project Title

this project is to play Azaan at the time of one's mosque bases on Mawaqit.
It is python script that runs on raspberrypi, it parses the mawaqit website and schedules the azaan to be played at the prayer time.



## Installation

1. Clone the repo
   ```sh
   git clone git@github.com:motazEmad/azaan-py.git
    ```
   
in azaan.py replace the url of mawaqit of your mosque
```
url = "https://mawaqit.net/en/{mosqueurl}"
```
